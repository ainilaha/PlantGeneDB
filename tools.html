BLAST

Basic Local Alignment Search Tool

BLAST (Basic Local Alignment Search Tool) finds regions of local similarity between sequences. The program compares nucleotide or protein sequences to sequence databases and calculates the statistical significance of matches.

JBrowse

Genome Browser

JBrowse is a fast, interactive genome browser built with HTML5 and JavaScript. It allows you to explore genomic data and annotations with unprecedented speed and responsiveness.

Key Features:

Fast and responsive visualization of large genomic datasets
Explore gene structures, variations, and functional annotations
Supports multiple reference genomes and coordinate systems
Customizable display and track configuration
